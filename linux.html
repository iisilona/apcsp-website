Command Name Description Usage
nano (file name) Opens the file editor for the given file. nano index.html to open the website editor!
mkdir (name) Creates a directory with the given name mkdir Hi Friends would make a directory called Hi Friends.
git init Makes empty git hub Exactly what it says on the tin
git add [file] Adds a file to the git staging area. git add Hi Friends to add the file Hi Friends
git commit -m "what i did" Commits all files in the git staging area to the git repository git commit -m "Added 5 files"
git status Shows all files in the staging area Exactly what it says on the tin
git log Shows a log of gits Exactly what it says on the tin
git remote add origin git@github.com:username/repository-name.git Adds an origin file with the given git url. git remote add origin git@github.com:iisilona/csp-website.git
git push -u origin master Pushes the origin and completes the remote add command Exactly what it says on the tin
ls [directory name] Lists directory contents ls Hello Friends would show the contents of file Hello Friends
